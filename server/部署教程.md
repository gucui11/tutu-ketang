# 腾讯云函数 SCF 部署教程（5分钟搞定）

## 一、注册腾讯云

1. 打开 https://cloud.tencent.com 注册/登录
2. 完成**实名认证**（需要身份证，1分钟搞定）

## 二、创建云函数

1. 登录后，顶部搜索框搜索 **「云函数」** 或 **「SCF」**，点击进入
2. 左侧选择 **「函数服务」**
3. 点击 **「新建」** 按钮
4. 填写配置：
   - **创建方式**：从头开始
   - **函数名称**：`tutu-ai-proxy`（随便取）
   - **地域**：选离你最近的（如「广州」或「上海」）
   - **运行环境**：**Node.js 18.15**（重要！）
   - **时延配置**：内存 256MB（够用）
   - **执行超时时间**：**60 秒**（AI 生成需要时间）
   - **请求处理程序**：`tutu-scf.handler`（index文件名的.handler，或者你粘贴代码时的文件名）
5. 点 **「下一步」** 或 **「完成」**

## 三、粘贴代码

1. 进入刚创建的函数 → 点击 **「函数代码」** 标签页
2. 你会看到一个编辑器，**清空默认代码**
3. 打开项目中的 `server/tutu-scf.js` 文件，**全部复制粘贴**进去
4. 点击 **「保存」** 或 **「部署」**

> 代码中已经包含了你的 DeepSeek API Key，家长看不到。

## 四、添加触发器（让外部可以访问）

1. 在函数详情页，找到 **「触发器管理」** 标签
2. 点击 **「创建触发器」**
3. 配置：
   - **触发方式**：**API网关触发器**
   - **请求方法**：**POST**（勾选 OPTIONS 也可以，用于跨域预检）
   - **鉴权方式**：**免鉴权**（重要！选这个，否则家长访问不了）
   - **安全域名**：留空
4. 点击 **「提交」**

## 五、获取访问地址

创建触发器后，会显示一个 **访问路径（URL）**，类似：

```
https://service-abc123.gz.apigw.tencentcs.com/release/tutu-ai-proxy
```

**把这个 URL 发给我**，我帮你填进前端代码，重新部署就能用了。

## 常见问题

**Q: 触发器里看不到 URL？**
A: 创建触发器后，点击触发器名称，详情页里有「访问路径」。

**Q: 访问报错？**
A: 确认触发器鉴权方式选的是「免鉴权」，确认运行环境是 Node.js 18.15。

**Q: 代码文件名不对？**
A: 腾讯云函数默认入口是 `index.handler`。如果你粘贴的文件叫 `tutu-scf.js`，
   需要在函数配置里把「执行方法」改成 `tutu-scf.handler`。
   或者把代码粘贴到默认的 `index.js` 文件里。
